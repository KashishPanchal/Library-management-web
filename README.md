# Library Management Web

A simple client-side library management system built with HTML, CSS, and JavaScript.

## Contents

- `library 1.html` - Main web page for the library management system.
- `txt.txt` - Additional project file.
- `kashish report.pdf` - Project report document.

## Features

- Book browsing interface
- Login and registration UI
- Book details modal with issue/return actions
- Search and filter functionality
- CSV download for issued books
- Toast notifications for user actions

## How to use

1. Open `library 1.html` in a browser.
2. Use the navigation links to explore the app.
3. Register or login to access protected sections like Home, Books, and Contact.
4. Search or filter books and view details.

## Notes

- This project is currently static and runs entirely in the browser.
- No backend server or database is required to run the demo.
